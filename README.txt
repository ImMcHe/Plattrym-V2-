-- PLATTRYM --
-- HOW TO RUN THE GAME --

First step, clone the git repository.
Then, open the "Compiled" folder and run "Plattrym.exe"

Or, alternatively, download the ".zip" on github and extract it.
Then, open the "Compiled" folder and run "Plattrym.exe"

Note this method allows you to see the source code for the project.

-- WHAT HAPPENED TO PLATTRYM V1 --

Plattrym V1 was originally built with C++, and is recognized as a wakatak trojan for most virus detectors. This is built with C. Hopefully the virus detectors will not recognize this as a virus.
