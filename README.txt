-- PLATTRYM --
-- HOW TO RUN THE GAME --

First step, clone the git repository.
Then, open the "Compiled" folder and run "Plattrym.exe"

Or, alternatively, download the ".zip" on github and extract it.
Then, open the "Compiled" folder and run "Plattrym.exe"

Note this method allows you to see the source code for the project.
